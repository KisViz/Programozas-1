[⭐⭐⭐] 15. Tehenek (8 pont)

Józsi bácsi teheneit mindenki ismeri. Különleges tehenei vannak, amelyek 3 éves koruktól kezdve szülnek 1 új borjút (0 éves tehén), emellett örökké élnek. Kezdetben Józsi bácsinak 1 db 0 éves tehene van. Azt szeretnénk megtudni, hogy az n. évben hány tehene lesz neki.
Példa:
év száma 0 éves 1 éves 2 éves 3- éves
0. év 1 db 0 db 0 db 0 db
1. év 0 db 1 db 0 db 0 db
2. év 0 db 0 db 1 db 0 db
3. év 1 db 0 db 0 db 1 db
4. év 1 db 1 db 0 db 1 db
5. év 1 db 1 db 1 db 1 db
6. év 2 db 1 db 1 db 2 db
7. év 3 db 2 db 1 db 3 db
8. év 4 db 3 db 2 db 4 db

Készítsd el a tehenszam metódust, ami egy paramétert vár: az évet. A metódus adja vissza, hogy az n. évben Józsi bácsinak hány tehene van összesen. Lehet, hogy nagy értékekre vagyunk kíváncsiak, így az eredményt String-ként kell visszaadni.

Maximális futásidő: 1 s
Ha nagy számokkal kell dolgoznunk, érdemes lehet használni a BigInteger osztályt!
