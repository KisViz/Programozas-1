[⭐⭐⭐] 27. Számok (6 pont)

Hozd létre a Szamok osztályt a Szamok.java fájlban, és abban a main függvényt. A parancssori argumentumok szavakat tartalmaznak, azt kell ellenőrizned, hogy ezek közül hány olyan szó van, ami csak számjegyeket tartalmaz. A megoldást írd ki a standard hibakimenetre.
A standard hibakimenetre a System.err.println utasítással tudunk kiírni.
Egy szöveg i. karakterét a stringnek a charAt() metódusával tudjuk elérni. Pl. "macska".charAt(2) eredménye 'c' lesz.
Java-ban is működik karakterek összehasonlítása. Pl. ha c egy char típusú változó: if (c >= 'a') { System.out.println("NAGYOBB"); }
