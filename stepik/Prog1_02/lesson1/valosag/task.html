[⭐⭐] 25. Valóság (4 pont)

Egy kis falu lakosai már régen azon töprengenek, hogy mitől valóságos egy naprendszer.
A lakos-gyűlésen az egyik tudósban felmerült egy ötlet, persze senki nem hitt neki. Szerinte:
"ha a csillagtól számítva a bolygók közül az első valahány kőzetbolygó, és az azután jövők gázbolygók (nem kőzetbolygók), tehát nem keverednek egymással, akkor az a naprendszer valóságos. Egyéb esetben nem valóságos."
A valosagos függvény eszerint tesztel egy naprendszert. 1 paramétere van: egy tömb, ami a bolygókat tartalmazza.

A függvény térjen vissza igazzal, ha a naprendszer valóságos, hamissal, ha nem.

Néhány példa (az true jelöli a kőzetbolygót, a false a gázbolygót. A nyíl után a visszaadott érték látható):
  • true, true, true, false, false –> true
  • true, false, false, false, false, false –> true
  • true, true, true –> true
  • false, false, false –> true
  • true, true, false, false, true –> false
  • true, false, false, true, true –> false
A Bolygót egy osztály, aminek az alábbi adattagjai vannak:
  • nev: a bolygó neve (szöveg)
  • tomeg: a bolygó tömege (egész)
  • kozetbolygo: a bolygó kőzetbolygó-e
Az osztályokra egyelőre gondolhatunk úgy, hogy struktúrák, néhány plusz funkcionalitással kiegészítve.
Ha egy boolean típusú változó értékét szeretnénk vizsgálni, akkor nem kell az ==-t használni, önmagában is egy teljes értékű kifejezés. Például (bolygok[i].kozetbolygo == true) helyett írjunk (bolygok[i].kozetbolygo)-t.
